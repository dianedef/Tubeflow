# Component Inventory (Web)

## Common Components

`apps/web/src/components/common/`

- **Logo**: Application logo component
- **Menu**: Navigation menu
- **UserNav**: User profile navigation/dropdown
- **Avatar**: User avatar display
- **Button**: Reusable button component
- **DropdownMenu**: Generic dropdown menu

## Home Page Components

`apps/web/src/components/home/`

- **Hero**: Landing page hero section
- **Benefits**: Feature highlights section
- **Testimonials**: User testimonials carousel/grid
- **Footer**: Application footer
- **ComplexToggle**: UI toggle for complex interactions

## Note Components

`apps/web/src/components/notes/`

- **Notes**: Main notes list container
- **NoteItem**: Individual note card/row
- **NoteDetails**: Full view of a note
- **CreateNote**: Form to create new note
- **DeleteNote**: Delete confirmation/action
- **Checkbox**: Custom checkbox component
