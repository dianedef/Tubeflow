# Component Inventory (Native)

## Screens

`apps/native/src/screens/`

- **LoginScreen**: Authentication entry point
- **NotesDashboardScreen**: Main list of user notes
- **CreateNoteScreen**: Form to add new notes
- **InsideNoteScreen**: Detail view for a specific note

## Assets

- Fonts: Inter, Montserrat families
- Icons: Custom PNG icons (Logo, Avatar, Save, Summary, etc.)
