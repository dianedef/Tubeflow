# Tubeflow Documentation

## Project Overview

- **Type:** Monorepo with 3 parts
- **Architecture:** Full Stack (Web + Mobile + Serverless Backend)

### Quick Reference

#### Web (web)

- **Type:** Web Application
- **Stack:** Next.js, React, Tailwind CSS
- **Root:** `apps/web`

#### Native (native)

- **Type:** Mobile Application
- **Stack:** React Native, Expo
- **Root:** `apps/native`

#### Backend (backend)

- **Type:** Backend Service
- **Stack:** Convex, TypeScript
- **Root:** `packages/backend`
